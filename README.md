## T-SQL-Cleaning-Housing-Data
### Transformed table of data using Microsoft SQL Server Management Studio

For this project I transformed a table of housing data using T-SQL in Microsoft's SQL Server Management Studio.

Techniques used in this project include: Self-Joins, Window Functions (ROW_NUMBER), CTEs, String Manipulation, CASE Statements.

As per usual, I transformed the data non-destructively, but this time I added code to Alter and Update the table.
The code is commented out and is mostly to demonstrate the knowledge of destructively changing, adding, and removing data, if ever necessary.
